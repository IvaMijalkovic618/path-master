# PathMaster

PathMaster is a grid-based Java game where the player navigates from a start point to an end point by selecting adjacent cells.

The game tracks the path taken, calculates scores based on visited values, and provides real-time feedback through a graphical user interface.

## Features
- Grid-based movement system
- Start and end point mechanics
- Path tracking and scoring system
- Timer and live game statistics
- Customizable grid size
- Game state saving to CSV file
- UI customization (colors)

## Technologies
- Java
- Swing (GUI)
